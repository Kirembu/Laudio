About Laudio

Laudio is a webclient and server for your Musiccollection. It aims to replace the DAAP standard which originates from Apple. After some years, Apple decided to encrypt iTunes' DAAP service making it nearly impossible to set up shares or listen to shares.

Other than Ampache, Laudio implements an easy to use interface and uses some of the web's newest features such as the HTML5 audio and tag canvas. This requires the users to have a recent browser which supports HTML5.

Laudio supports MP3 and Ogg Vorbis, both implemented using HTML5. That means if you want to listen to MP3, you need a browser which is able to play MP3, such as Chromium. Firefox only supports Ogg Vorbis.

Recommended Browser

  • Firefox 3.5 or higher
  • Chrome 4 or higher
  • Chromium 5 or higher

Less recommended webbrowser

  • Epiphany: extremely slow
  • Apple Safari - Safari only plays MP3

Not recommended browsers

  • Microsoft Internet Explorer any version - Doesnt support HTML5 yet

I didnt try Konqueror or Internet Explorer 9 yet, please send me an email so i can update this description.
