About Laudio

Laudio, whose name orignites from LAN Audio, is a webclient for your Musiccollection. It aims to replace the currently mostly broken DAAP players, which can stream DAAP shares over the local network. Thanks to Apple, who break/encrypt their services after external programs start using it, DAAP is partly broken ever since including iTunes 7.

Future Releases also plan some filesharing service for you and your friends (only legal tracks should be shared -> jamendo.com) and an access control to your collection. Maybe we'll support ampache xml interfaces too if it isnt too hard ;)

Recommended webbrowser are:

  • Google Chrome/Chromium v. 4+ - If you dont like Google, just use it locally, defenitely the fastest and best choice though
  • Firefox 3.5+ - A bit slow due to some bugs with large table which we are using heavily

Less recommended webbrowser are:

  • Konqueror 4.4+ - Well, I didnt test it but it should run quite well. You'll need the codecs from your gstreamer/xine backend though
  • Epiphany 2.30+ - Supporting audio/video but "support is not perfect"

Not recommended are the following ones due to these issues:

  • Microsoft Internet Explorer any version - Slow and no HTML 5, you really dont want this one
  • Apple Safari - Were replacing an Apple product, so yes, Safari is of course the only browser who doesnt support ogg in its HTML5 audio/video tags

Known Issues

  • Slow when having huge (10k+ songs) collections
  • Slow in Firefox when displaying a lot of songs: this is a Firefox bug occuring with large tables
  • Lacking mp3 support (think of it as feature, not a bug)
