About

About

Ever wanted to share your music over your network or the Internet? Your iTunes shares dont work with your music player? Ampache User Interface gives you headaches? Behold! We got just what you needed: Some new software called Laudio will fix your issues.

Instead of relying on Apple's meanwhile broken (Apple encrypts its iTunes shares) DAAP standard, Laudio aims to deliver a standard solution based on the HTML5 standard. Laudio is completely written in Python and Javasript and can be used from any fast enough and up to date Browser.

Laudio supports Ogg Vorbis and Mp3 audio files, but in case of Vorbis only for the browsers which support it. Vorbis is being played using the HTML5 audio tag, Mp3 is being played by a flash plugin.

Download

We got a PPA for Maverick: Just type in

$ sudo apt-add-repository ppa:bernhard-posselt/laudio-ppa
$ sudo apt-get update
$ sudo apt-get install laudio

If you're feeling lucky, you can try our highly experimental setup script, which provides installation for Arch Linux, Gentoo and Ubuntu.

More Information is Available on the Github page.
